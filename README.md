# FAQ Chatbot

## Overview
This project is a Python-based FAQ Chatbot that answers user questions using Natural Language Processing techniques. It compares the user's query with a predefined FAQ dataset and returns the most relevant answer.

## Features
- Interactive web interface
- Question-answer matching
- Fast response generation
- User-friendly design

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

## Installation

pip install streamlit pandas scikit-learn

Run the Project: streamlit run chat.py
